# 🚗 Gustz Cars - Product Slider Landing Page

Uma landing page moderna com slider animado, desenvolvida utilizando HTML, CSS e JavaScript puro.

O projeto simula a apresentação de carros esportivos premium, com foco em design moderno, animações suaves e experiência visual imersiva.

---

## 🚀 Preview

🔗 Acesse online:  
https://gustz7.github.io/product-slider---landing-page/

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3 (Animações, Flexbox, Transitions)  
- JavaScript (Manipulação de DOM e Slider)  
- Git & GitHub  

---

## ✨ Funcionalidades

- Slider com transição lateral animada  
- Botões de navegação (Next / Previous)  
- Indicadores dinâmicos de slide  
- Animações suaves com `transform` e `opacity`  
- Efeito hover interativo nos botões  
- Layout moderno com estética tech  

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação de layout responsivo  
- Animações avançadas com CSS  
- Manipulação de classes com JavaScript  
- Organização de código front-end  
- Controle de versionamento com Git
